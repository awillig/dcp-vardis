Version 1.1, October 2024
=========================

This is an update to the version 1.0 release with significant clarifications
to the specification (while leaving packet formats unchanged) and corresponding
changes made to the reference simulator.


Version 1.0, September 2024
===========================

This is the first release. It contains the specification of the
DCP/VarDis protocol stack and a simulator based on the OMNeT++
discrete-event simulation framework.
